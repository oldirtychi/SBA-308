# SBA-308
Step by step this script is meant to:

1. Check if a course ID is valid or not

2. Calculate the percentage of points and to throw an error if it is 0

3. check whether the submission date is past the due date

4. gather data from the assignment id, date sumbitted, and final score of the assignment

5. determine wether an assignment is due yet

6. deduct 10% of points if an assignment was submitted past the due date

7. calculate the average points of each learner

8. log the data as an array
